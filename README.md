# powerbi-sales-dashboard
Project Overview
This project analyzes sales performance using an interactive dashboard built in Power BI. The dashboard provides insights into revenue, profit, regional performance, and product categories to support data-driven business decisions.

Objectives
	•	Track total sales and total profit
	•	Measure profit margin
	•	Identify top performing regions and categories
	•	Analyze sales trends over time

Tools Used
	•	Power BI
	•	DAX (Data Analysis Expressions)
	•	Data visualization techniques

Key Features
	•	KPI cards for Sales, Profit, and Profit Margin
	•	Sales vs Profit comparison chart
	•	Regional performance analysis
	•	Category distribution visualization
	•	Interactive filters (Region and Year)

Insights
	•	Sales increased significantly between 2015 and 2016
	•	Some regions generate high sales but lower profit margins
	•	Certain product categories contribute more to overall revenue

Outcome
This dashboard helps stakeholders quickly monitor performance and identify growth opportunities.
